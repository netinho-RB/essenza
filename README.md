Uma landing page institucional desenvolvida para a **Essenza Clinic**, focada em transmitir sofisticação, confiança e modernidade. O projeto utiliza conceitos de design minimalista e **Glassmorphism**.

[Link para o Projeto ao Vivo (se tiver)](https://seu-link-vercel.app)

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de front-end:

* **HTML5** - Estrutura semântica.
* **CSS3 Moderno** - Variáveis (Custom Properties), Flexbox, CSS Grid e Media Queries.
* **JavaScript Vanilla** - Lógica de menu mobile, carrossel 3D e efeitos de scroll.
* **Google Fonts** - Tipografia luxuosa com *Playfair Display* e *Inter*.
* **FontAwesome** - Iconografia premium.

---

## ✨ Diferenciais do Projeto

- [x] **Mobile-First & Totalmente Responsivo:** Experiência otimizada de 320px até UltraWide.
- [x] **Estética Minimalista:** Uso estratégico de espaços em branco e tons terrosos.
- [x] **Efeitos de Revelação (Reveal):** Animações suaves durante o scroll para uma experiência imersiva.
- [x] **Menu Glassmorphism:** Interface flutuante com desfoque de fundo em dispositivos móveis.
- [x] **Botão Flutuante WhatsApp:** Com animação de pulso para aumentar a taxa de conversão.
